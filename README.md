# 記帳應用程式(DailyAccount)
(此專案目前還在持續更新中...)

## 📖 專案描述
- 這是一個使用C# MVVM+MVP架構開發的學生管理系統，小型的WPF專案。
- 想要做到主要功能運作良好及邏輯與介面分離，但畫面陽春的情況下，刪除整個Xaml檔，由AI重新生成一個任何風格的介面Xaml檔，能夠直接套用且功能都正常使用。

## 🚀 主要功能
- 
- 自製CSV檔案讀取/寫入檔案

## 🏗️ 技術架構
- **語言**：C#
- **UI框架**：WinForm
- **框架**：.NET Framework
- **架構**：MVP
- **設計模式**：獨體模式(Singleton Pattern)
- **記憶體洩漏問題**：
  - 自製DebounceTime
  - Image圖檔壓縮及調整大小
- **資料處理**：自製CSV Reader/Writer Library
