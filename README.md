# 記帳應用程式(DailyAccount)
(此專案目前還在持續更新中...)

## 📖 專案描述
- 這是一個記帳應用程式，採MVP架構設計，透過使用者的每日記帳記錄，可隨時查詢單日或日期區間的所有支出記錄，並根據支出內容統計成圖表分析及帳戶分析。

## 🚀 主要功能
- 每日記帳支出記錄(日期、金額、類型、細項、對象、支付方式等等)
- 上傳收據或圖片記錄
- 查詢日期區間支出記錄
- 支出內容統計圖表(圓餅圖、直條圖、折線圖)

## 🏗️ 技術架構
- **語言**：C#
- **UI框架**：WinForm
- **框架**：.NET Framework
- **架構**：MVP
- **設計模式**：獨體模式(Singleton Pattern)
- **記憶體洩漏問題**：
  - 自製DebounceTime : 使用在記帳及查詢按鍵，避免連續觸發造成記憶體洩漏
  - Image圖檔壓縮及調整大小 : 4K圖檔壓縮至10KB以下
- **資料處理**：自製CSV Reader/Writer Library
- **類別轉換**：Automapper
